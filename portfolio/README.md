# 📊 Abdullah's Data Analyst Portfolio

A bilingual (English/Arabic) interactive portfolio built with **Streamlit** showcasing data analysis projects.

## 📁 Projects Included

### 1. 🌤️ Egyptian Weather Dashboard
- Real-time weather data for Egyptian governorates
- 7-day forecast, air quality, rain probability
- **Tech:** Python, Streamlit, Weather API

### 2. 🛒 E-Commerce Sales Analysis
- Sales performance across regions and categories
- Revenue trends and category insights
- **Tech:** Power BI, DAX, Data Modeling

### 3. 👥 HR Analytics Dashboard
- Employee turnover analysis across departments
- Salary distribution and retention insights
- **Tech:** Power BI, DAX, HR Analytics

## 🚀 How to Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/portfolio.git
cd portfolio

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
streamlit run portfolio.py
```

## 🌐 Deploy on Streamlit Cloud

1. Push code to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub repo
4. Select `portfolio.py` as the main file
5. Click Deploy!
